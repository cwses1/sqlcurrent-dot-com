Setup / Install
========================================================================================================================



Python
---------------------------------------------

SQL Current is written in Python.  You will need to get Python.

https://www.python.org

ANTLR Python Runtime
---------------------------------------------
::

	pip install antlr4-python3-runtime

Python Database Drivers / Adapters
---------------------------------------------

::

	pip install psycopg
	pip install pymssql
